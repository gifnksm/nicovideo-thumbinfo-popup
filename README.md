nicovideo-thumbinfo-popup
=========================

UserScript to get nicovideo movie information before going to watch page.

This branch is in a very early stage of development.

## Setup

```
$ cd ./<path to cloned directory>
$ npm install
```

## Build

```
$ gulp build
```

## Test

```
$ gulp test
```

To start test watcher:

```
$ gulp watch
```
